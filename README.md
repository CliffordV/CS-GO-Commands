# CS-GO-Commands

### ez pz lemon sqz

* Download as zip
* Extract
* Open knife_binds folder
* Move all files in this path 
* "...\Steam\steamapps\common\Counter-Strike Global Offensive\csgo\cfg"
* Run CS:GO
* Console and type "exec knife_binds"
* You can also type "exec knife_unbindall" to unbind
